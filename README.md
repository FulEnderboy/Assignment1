# MSR Knowledge Base (COIS 3060H — Assignment 1)

![CI](https://github.com/FulEnderboy/Assignment1/actions/workflows/ci.yml/badge.svg)

A simple single-page web application that serves as a knowledge base for Mining Software Repositories (MSR) research. It helps new MSR students quickly understand:
- Data Sources
- Collection Methods
- Analysis Methods
- Publication Venues

## Repository Structure
- `/docs/requirements.md` — requirements + user stories
- `/public` — HTML/CSS/JS implementation
- `/tests` — automated tests + manual display and resolution test
- `/.github/workflows/ci.yml` — CI pipeline

## How to Run
Open `/public/index.html` in a browser.
